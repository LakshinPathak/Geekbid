# GeekBid Protocol: Technical Documentation & Style Guide

## 1. Brand Identity
GeekBid is a high-fidelity Dutch auction platform for elite engineering talent. The brand balances "Cyber-Minimalist" precision with a "Royal & Rich" aesthetic for its premium light-mode experience.

### Visual Pillars:
- **Precision**: Grid-aligned layouts and monospaced accents.
- **Authority**: High-contrast typography and deep emerald tones.
- **Transparency**: Glassmorphic layers and real-time telemetry logs.

## 3. Core Mechanics
- **Dutch Auction**: Prices start at a ceiling and decay over time. The UI must emphasize the "Current Price" and "Decay Rate".
- **GeekScore™**: A proprietary reputation metric (0.0 - 99.9). Displayed with a specific "Verified" badge.
- **Escrow Protocol**: All payments are secured in on-chain vaults. Financial screens use heavy glassmorphism to denote security.

## 4. Component Standards
- **Buttons**: Rounded-md (8px). Primary buttons use Imperial Emerald with white text.
- **Cards**: Soft 1px borders (#E5E7EB) with subtle drop shadows (blur: 20px, opacity: 5%).
- **Terminal Logs**: Monospace font (JetBrains Mono or similar) on a soft gray background.
