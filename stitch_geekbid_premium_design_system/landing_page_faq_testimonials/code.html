How do Dutch auctions function on GeekBid?

expand_more
Our Dutch auctions start at a ceiling price and decrease linearly over a set duration. This mechanism ensures fair discovery and allows participants to enter at the price point that matches their valuation. Once the reserve is hit or the supply is exhausted, the auction concludes instantly.

What is GeekScore and how is it calculated?

expand_more
GeekScore is a dynamic reputation metric (0-100) derived from on-chain activity, GitHub contributions, and historical bidding integrity. A high GeekScore grants priority access to tier-1 auctions and reduced protocol fees. It's designed to prioritize long-term builders over mercenary capital.

Is my capital secure in Escrow?

expand_more
All funds are held in non-custodial smart contracts audited by Tier 1 security firms. The escrow logic uses a dual-signature mechanism: release is only triggered upon auction fulfillment or expiration-refund. GeekBid never has direct access to user private keys or collateral.

Can I integrate the bidding API into my own tools?

expand_more
Yes. We provide a robust WebSocket and REST API for programmatic interaction. This allows power users to build custom front-ends or automated bidding strategies. Documentation is available in our 'Vault' section for verified wallet holders.
